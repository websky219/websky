使用CSS3制作文字、图片倒影

text-shadow

说明：text-shadow设置文本阴影。text-shadow 可向文本应用阴影。您能够规定水平阴影、垂直阴影、模糊距离，以及阴影的颜色

语法：text-shadow：none (默认值)

示例：

h1

{

text-shadow: 5px 5px 5px #FF0000; /* 第1个5px是阴影水平偏移值，第2个长度值是阴影垂直偏移值 ，第3个阴影模糊值*/

}

(1)普通文字阴影

测试普通文字阴影效果

text-shadow:2px 2px #0F0

(2)模糊文字阴影效果

测试模糊文字阴影效果

text-shadow:2px 2px 1px #0F0

(3)多重模糊文字阴影效果

测试多重模糊文字阴影效果

text-shadow:2px 2px 0 #0F0,1px 1px 2px #fff;
